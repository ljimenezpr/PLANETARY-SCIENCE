# PLANETARY-SCIENCE

Este repositorio reúne material de programación y talleres prácticos enfocados en **ciencias planetarias y geología**.  
El objetivo es facilitar el aprendizaje y la exploración de datos geoespaciales, modelos digitales de elevación y técnicas de análisis aplicadas al estudio de volcanes, tectónica y superficies planetarias.

## Contenido
- **Talleres en Python**: cuadernos diseñados para ejecutarse en **Google Colab**, con ejemplos reproducibles y explicaciones paso a paso.  
- **Recursos de datos**: archivos geoespaciales y modelos digitales de elevación utilizados en los talleres.  
- **Visualizaciones 3D y mapas**: ejemplos de cómo generar representaciones interactivas de terrenos y texturas.

## Uso
1. Abra el cuaderno correspondiente en Google Colab.  
2. Copie y pegue el código, o cargue directamente el archivo `.ipynb` en Colab.  
3. Ejecute las celdas siguiendo las instrucciones para reproducir los análisis y visualizaciones.
